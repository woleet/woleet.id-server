Delete user identity CommonName?

Warning: deleting a user identity is irreversible, and also deletes all associated keys and API tokens.

Consequently, the user won’t be able to sign anymore. The validation of signatures made precedently will be solely based on the signature proof receipt information:

  • The signatory's identity will be that of the signature proof receipt.
  • The verification of key expiration and revocation status will no longer be possible.

Delete seal identity CommonName?

Warning: deleting a seal identity is irreversible, and also deletes all associated keys and API tokens.

Consequently, seal creation won’t be possible anymore for this seal identity. The validation of seals made precedently will be solely based on the seal proof receipt information:

  • The seal's identity will be that of the seal proof receipt.
  • Key ownership will no longer be provable by the identity server.
  • The verification of key expiration and revocation status won’t be possible anymore.
